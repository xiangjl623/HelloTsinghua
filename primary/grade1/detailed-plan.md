# 小学一年级详细计划（2026.9 – 2027.7）

> 学校：中关村二小百旺校区  
> 孩子特点：语文 / 阅读 / 表达**显著超前**；数学**已完成一年级《举一反三》A 版**；英语**相对薄弱、不太敢说**  
> 家庭模式：双职工，平日轻量、周末系统投入  
> 核心原则：**养习惯 > 抢跑知识**；配合学校零起点，不刷奥数、不背词表

**配套文件**：[成长档案](../../shared/templates/growth-archive.md) · [专注力训练](../templates/focus-training-games.md) · [暑假衔接](../templates/summer-2026-checklist.md) · [周检查表](../templates/grade1-weekly-checklist.md)

**培养素材**：[语文](materials/chinese.md) · [数学](materials/math.md) · [英语](materials/english.md) · [兴趣总览](materials/interests.md) · [游泳](materials/swimming.md) · [跳绳](materials/jump-rope.md) · [音乐](materials/music.md) · [游泳进度](swim-progress.md)

**英语基线**：相对薄弱 · **不太敢说** → [english.md](materials/english.md)（先听敢说，不纠音过狠）

## 一、学年总目标

| 维度 | 学年结束时应达到 |
|------|------------------|
| 习惯 | 作息稳定、书包书桌自理、放学「先任务后玩」成自然 |
| 情绪 | 喜欢上学，分离焦虑消失，愿意和老师同学交流 |
| 专注 | 课堂能连续听讲约 25–30 分钟（见专注力训练 Level 4） |
| 语文 | 每日**自主阅读**；三国 / 水浒等讨论表达；诗词复习 + 适量新增；**写字随校内** |
| 数学 | 跟住校内；棋类 / 数独 / 生活数学；**无畏难、不重复刷一年级举一反三** |
| 英语 | **听懂**课堂指令；**愿意**跟唱 / 回声 2–3 句；敢开口比发音准更重要 |
| 体育 | **游泳**、**跳绳**、**音乐**（见 [兴趣总览](materials/interests.md)） |
| 不做 | 奥数、提前学二年级课本、英语单词表、每天写字超 15 分钟、**重刷一年级举一反三** |

## 二、学年节奏概览

```
8月        9月         10–12月        1月      2月        3–6月         7月
暑假衔接 → 适应期 → 第一学期建立期 → 寒假巩固 → 第二学期深化期 → 学年验收
```

| 阶段 | 时间 | 关键词 |
|------|------|--------|
| 入学前 | 2026.8 | 作息、棋类 / 数独 / 生活数学、亲子阅读、熟悉校区 |
| 适应期 | 9 月 | 分离焦虑、课堂规则、建立放学流程 |
| 建立期 | 10–12 月 | 习惯固化、阅读提速、专注力 Level 2–3 |
| 寒假 | 2027.1–2 | 阅读不断档、复习不写新、体育保持 |
| 深化期 | 3–6 月 | 自主阅读、看图说话、英语跟唱 / 回声、表达活动 |
| 验收 | 7 月 | 对照总目标复盘，准备二年级 |

### 9 月 · 适应期（开学月）

**本月重心**：情绪稳定 > 学习进度。前 3 周不**新报**课外班；**已在上的游泳课可续**（重点打腿）。

#### 第 1 周（开学周）

| 天 | 家长重点 |
|----|----------|
| Day 1–2 | 提前 10 分钟到校；固定接送人、固定地点；回家只问「今天有什么开心的事」 |
| Day 3–5 | 观察作业量；建立「回家 → 洗手加餐 → 休息 15 分钟 → 学校任务 → 玩」流程 |
| 周末 | 不参加新班；户外 + 自由玩耍；9 月月日填 [成长档案](../../shared/templates/growth-archive.md) |

#### 第 2–4 周

| 学科 | 每日家庭时间 | 具体做法 |
|------|--------------|----------|
| 语文 | 30 分钟 | **自主阅读**《三国》《水浒》等；读后讨论 5 min（见 [chinese.md](materials/chinese.md)） |
| 数学 | 0–10 分钟 | **仅配合**学校内容；用水果、积木复习当天学的数 |
| 英语 | 15–20 分钟 | 磨耳朵为主；TPR / 跟唱，**不强求跟读** |
| 专注 | 每周 3 次 × 10 分钟 | [听指令游戏](../templates/focus-training-games.md) Level 1 |
| 游泳 | 续课 1 次 / 周 | 向教练问打腿反馈；周末岸上直腿 2 次 × 5 min |

#### 9 月家长必做

- [ ] 新生家长会：记录班主任姓名、电话、班级群规、作业政策
- [ ] 确认上下学路线、午餐/加餐安排
- [ ] 与班主任一次简短沟通：「孩子在家阅读较多（已读名著少儿版），**校内写字写话完全配合零起点**」
- [ ] 建立成长档案与 [游泳进度表](swim-progress.md)

#### 9 月验收

- [ ] 连续 2 周 21:00 前入睡
- [ ] 能说出班级里 2 个朋友名字
- [ ] 整理书包基本不用催
- [ ] 对上学无持续哭闹或拒学
- [ ] 游泳续课正常；打腿有教练反馈记录

### 10 月 · 习惯固化启动

**本月重心**：固定流程；阅读以**自主 + 讨论**为主（已能独立读无拼音读本，不必回退桥梁书）。

| 学科 | 频率 | 内容 |
|------|------|------|
| 语文 | 每日 30–40 min | 自主读《三国》《水浒》等；读后讨论 5 min |
| 诗词 | 每周 4–5 天 × 5 min | 复习已背二、三年级篇目 |
| 数学 | 每周 3 次 × 10 min | 购物算账、分水果；配合校内 |
| 英语 | 每日 20 min | Peppa / 牛津树；**跟唱或 TPR**，跟读可选 |
| 专注 | 每周 3–4 次 | Level 1 → Level 2 |
| 表达 | 每周 1 次 | 说书 5 min（三国 / 水浒片段） |

#### 10 月周末

| 周六 | 周日 |
|------|------|
| 上午：户外 2h | 上午：图书馆选书（三国 / 历史类） |
| 下午：说书 + 听指令游戏 | 下午：家庭日 |

#### 10 月验收

- [ ] 放学流程无需口头提醒
- [ ] 能独立阅读无拼音课外书 30 min
- [ ] 能讲清 1 个三国或水浒人物的特点

---

### 11 月 · 第一学期中段

**本月重心**：配合期中前后校内的节奏；**不因考试加练**；巩固书写姿势。

| 学科 | 调整 |
|------|------|
| 语文 | 关注校内写字要求；家庭每日写字 **≤ 10 分钟**，只练当天生字，重姿势 |
| 数学 | 若校内有口算：每天 5 分钟口算，**限时但不排名** |
| 英语 | 可选「跟唱录音」1 次 / 周：唱 1 段儿歌即可（**不必读句子**） |
| 专注 | Level 2 稳定；尝试 Level 3（三步指令） |
| 表达 | 鼓励班级「讲故事」——可讲**三国 / 水浒**片段 |

#### 期中前后家长话术

- 考得好：「你上课认真听了吗？这个方法可以继续用。」
- 考一般：「才一年级，我们看看哪道题是听懂了但写错了。」
- **避免**：「别人家孩子考多少」

#### 11 月验收

- [ ] 握笔姿势基本正确（背无严重弯曲、纸不歪 45° 以上）
- [ ] 校测无论高低，孩子无严重畏难或哭泣
- [ ] 参加过 1 次班级或校级表达类活动（讲故事 / 朗诵 / 值日介绍均可）

---

### 12 月 · 第一学期收尾

**本月重心**：复习校内、保证睡眠；阅读带入寒假书单。

| 项目 | 做法 |
|------|------|
| 校内复习 | 跟着老师走；家庭只帮孩子**讲懂**错题，不额外刷卷 |
| 阅读 | 每日 30 分钟不断；选 2 本稍长桥梁书作为寒假预备 |
| 英语 | 学 2–3 首英文圣诞 / 节日儿歌（趣味为主） |
| 社交 | 如有班级活动、元旦表演，鼓励孩子参与 |
| 户外 | 冬天仍保证每周户外 ≥ 3 次 × 1 小时（跑步、骑车、爬山） |

#### 12 月验收

- [ ] 第一学期各科随校内进度，无严重掉队
- [ ] 三步指令能完成 70% 以上
- [ ] 确定寒假 5 本必读书单

### 1 月 · 寒假（上）

**本月重心**：**不断阅读、不写新课**；游泳 / 音乐寒假班可选

| 项目 | 建议 |
|------|------|
| 作息 | 可稍晚起（7:30），但 21:30 前睡；Travel 日也要读书 |
| 阅读 | 每日 ≥ 40 分钟；完成 3 本桥梁书 |
| 数学 | 生活场景：买年货算账、量身高体重记录 |
| 英语 | 磨耳朵 20 min / 天；**只听不说 OK** |
| 写字 | 暂停或每周 2 次 × 5 分钟复习旧字 |
| 体育 | 游泳 1 次 / 周（扶板打腿）；跳绳每日；音乐练习 |
| 专注 | 维持 Level 3，每周 2 次即可 |

#### 寒假书单建议（5 本）

1. 《了不起的狐狸爸爸》（罗尔德·达尔）
2. 《小屁孩日记》注音版（选 1 册，轻松）
3. 《西顿野生动物故事集》简版（1–2 篇 / 天，非虚构入门）
4. 《父与子》漫画版
5. 孩子自选 1 本

#### 1 月验收

- [ ] 阅读 5 本中完成 ≥ 3 本
- [ ] 20 以内口算速度未明显退步
- [ ] 游泳续课稳定；扶板打腿有进步感；音乐班确定；跳绳每日习惯建立

### 2 月 · 寒假（下）+ 开学准备

**本月重心**：收心；最后 5 天恢复 7:00 起床。

| 时间 | 动作 |

| 2 月 16–20 日 | 逐步提前起床；整理书包、文具；检查姓名贴 |
| 开学前 3 天 | 21:00 睡；模拟「上学日」时间表；心理预热：「又要见老师和同学了」 |

#### 2 月验收

- [ ] 开学前 3 天恢复 7:00 起床
- [ ] 书包文具齐全
- [ ] 对返校无强烈抗拒

### 3 月 · 第二学期启动

**本月重心**：从自主阅读过渡到**深度讨论 + 写话**；表达活动常态化。

| 学科 | 目标 |
|------|------|
| 语文 | 自主阅读 30 min + 讨论；《西游记》或历史类可加入；**3 句日记**每周 2 次 |
| 诗词 | 新增 3–5 首（与三国 / 历史相关优先） |
| 数学 | 配合校内；数独 4×4 每周 2 次 |
| 英语 | 回声 2–3 句或跟唱 1 首（**小声即可**） |
| 表达 | 校级朗诵 / 故事会（诗词或说书） |

#### 3 月阅读

- 继续三国 / 水浒；或换《西游记》《史记》故事少儿版
- **不必**回退注音版《夏洛的网》等（除非孩子自选）

#### 3 月验收

- [ ] 能独立阅读 15 分钟不分心
- [ ] 日记能口述 3 句完整话
- [ ] 五子棋或跳棋能对弈 1 局

### 4 月 · 春学期中段

**本月重心**：户外加大；看图说话 + 日记结合。

| 项目 | 做法 |

| 户外 | 圆明园 / 颐和园 / 香山（选近处）每月 2 次半天 |
| 英语 | 动画一集 + **可选**用中文讲 1 句剧情（英文 1 词也算） |

#### 4 月验收

- [ ] 日记能独立写 3 句以上
- [ ] 看图说话逻辑清楚（有开头、中间、结尾）

## 六、三科家庭配合要点

### 语文（优势学科 · 重点投入）

| 活动 | 频率 | 说明 |
|------|------|------|
| 亲子 / 自主阅读 | 每日 30–40 min | 自主读名著少儿版；读后讨论 5 min |
| 复述故事 | 每周 1 次 | 训练表达，为二年级写作打底 |
| 看图说话 | 10–12 月起每周 1–2 次 | 3 句 → 5 句 |
| 日记 | 3 月起每周 2 次 | 口述 → 独立写 |
| 写字 | 每日 ≤ 10 min | 只练校内字，重姿势 |
| 表达活动 | 每学期 ≥ 1 次 | 讲故事、朗诵、值日 |

**推荐阅读**（已能独立读无拼音读本）：

1. 9–12 月：《三国》《水浒》精读 + 讨论；诗词复习
2. 1–2 月：换《西游记》或历史类；家庭说书
3. 3–6 月：加 1 本现代 / 科普；日记 + 朗诵

详见 [chinese.md](materials/chinese.md)。

## 八、家长沟通手册

### 每天回家只问三句

1. 今天有什么开心的事？
2. lunch 吃了什么？（轻松话题）
3. 需要我帮你什么吗？

**不问**：「考了多少分」「谁比你考得好」「老师批评你了吗」

### 与老师沟通（每学期 1 次即可）

- 孩子在家阅读较多，会配合零起点
- 想了解：课堂专注、与同学相处、写字姿势是否需要纠正
- 感谢老师，不暗示「请多关照」

### 孩子说不愿上学时

1. 先接纳：「你有点不想去，是吗？」
2. 再询问：「是想念妈妈，还是有什么事不开心？」
3. 不讲大道理；承诺准时接
4. 持续 2 周以上 → 与班主任约聊

## 十一、常见问题

| 问题 | 建议 |
|------|------|
| 作业太少，要不要加练？ | 不加；用阅读 + 户外 + 棋类填满；**不必重刷举一反三** |
| 识字比同学慢？ | 阅读已超前；**写字随校内**，不必在家超前识字 |
| 校内数学太简单？ | 正常；守课堂规矩比加难度重要；回家玩数独 / 棋类 |
| 数学校测偏低？ | 查粗心与读题；生活数学 + 讲错题；三年级再评估是否加班 |
| 老人带娃不一致？ | 统一「先任务后玩」一条即可 |
| 想报写字班？ | 一年级不报；在家 10 min / 天足够 |
| 同学都在上学而思？ | 一年级不比较；我们 3 年级再评估 |
| 英语不太敢说？ | **正常**；先听 3 个月 + 跟唱 / TPR；不纠音、不报班；见 [english.md](materials/english.md) |
