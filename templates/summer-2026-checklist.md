# 2026 暑假行动清单 · 入学前 3 个月

> 目标：平稳衔接中关村二小百旺一年级，**不抢跑刷题**。  
> 适用：2026 年 6 – 8 月（9 月入学）

---

## 总目标

- [ ] 8 月起按小学作息生活
- [ ] 20 以内加减法熟练
- [ ] 认常用汉字约 500 个（不必 2500）
- [ ] 能听懂并执行 3 步口头指令
- [ ] 每日亲子阅读 30 分钟成习惯
- [ ] 走访校区周边，建立期待感
- [ ] 游泳续课（自由泳 · 重点打腿）；记录于 [进度表](../grade1/swim-progress.md)

---

## 六月：观察与轻启动

### 作息（尚未强制，但开始靠拢）

- [ ] 记录当前睡眠时段，评估与 21:00 睡的差距
- [ ] 开始固定**睡前亲子阅读** 20 分钟（任意绘本）

### 能力摸底（不做书面测试）

- [ ] 口算：10 以内加减是否流畅？
- [ ] 识字：生活中认店名、路牌，好玩即可
- [ ] 指令：玩 [听指令游戏](../templates/focus-training-games.md) Level 1

### 阅读书单（规则类 · 入学心理准备）

- [ ] 《大卫上学去》
- [ ] 《小阿力的大学校》
- [ ] 《一年级的大豆》或同类
- [ ] 《魔法亲亲》（分离焦虑可选）

### 实地

- [ ] 周末路过百旺校区，外观打卡，告诉孩子「这是你九月要来的地方」

---

## 七月：系统启动

### 作息

- [ ] 7 月中旬起：7:30 前起床，21:00 前入睡（逐步提前，勿一步到位）
- [ ] 固定三餐时间，与将来上学节奏接近

### 数学（每日 10–15 分钟）

- [ ] **棋类 / 数独**：五子棋、4×4 数独、购物算账
- [ ] 口算：若已流畅则**不必冲刺**（已完成一年级《举一反三》A 版）
- [ ] **禁止**：奥数、一年级课本系统预习、重刷一年级举一反三

### 语文

- [ ] 亲子阅读 **30–40 分钟 / 天**
- [ ] 常用字 500：生活场景认读 + 简单字卡（每天 5 字即可）
- [ ] 每周让孩子**复述**一个故事

### 英语

- [ ] 磨耳朵 20 分钟 / 天：英文动画或绘本音频
- [ ] 不要求跟读、不要求背单词

### 自理训练

- [ ] 自己穿脱衣服、系鞋带（魔术贴鞋可先过渡）
- [ ] 整理玩具、收拾小书包
- [ ] 认识自己的姓名（口头 + 书写尝试）

### 游泳（续课 · 自由泳打腿）

- [ ] 维持 **1 次 / 周** 续课（已有约 15 节基础）
- [ ] 每节向教练问 1 句打腿反馈
- [ ] 周末岸上直腿模仿 2 次 × 5 min（见 [swimming.md](../grade1/materials/swimming.md)）
- [ ] 填写 [游泳进度表](../grade1/swim-progress.md)

### 户外

- [ ] 每周至少 2 次半天户外：百望山、圆明园、颐和园外圈步行

## 八月：冲刺适应（按小学时间表）

### 作息（8 月 1 日起）

| 项目 | 目标 |
|------|------|
| 起床 | 7:00 |
| 入睡 | 21:00 |
| 午睡 | 可选，30 分钟，8 月下旬逐步取消 |

### 每日日程模板

| 时段 | 内容 |
|------|------|
| 7:00 – 8:00 | 起床、早餐、简单家务 |
| 8:30 – 9:00 | 数学游戏 / 口算 |
| 9:00 – 9:40 | 亲子阅读 |
| 9:40 – 10:00 | 休息 |
| 10:00 – 10:20 | 英语磨耳朵 |
| 10:20 – 11:00 | 专注力游戏（听指令） |
| 上午其余 | 户外 |
| 15:00 – 15:30 | 自由阅读或休息 |
| 15:30 – 16:00 | 识字 / 复述故事 |
| 16:00 后 | 游泳续课 / 岸上直腿练习；音乐可选 |
| 20:00 – 20:30 | 洗漱、睡前阅读 |
| 21:00 | 入睡 |

### 能力验收（8 月底）

- [ ] 20 以内加减，口算 20 题 ≤ 5 分钟，正确率 ≥ 90%
- [ ] 常用字 500 左右（不要求会写全部）
- [ ] 完成 3 步连续指令 5 组无错误
- [ ] 连续亲子阅读 30 分钟不分心
- [ ] 独立整理小书包
- [ ] 游泳：扶板打腿较开学前有**进步感**（自评或教练反馈）

### 入学准备物品

- [ ] 书包（轻、有反光条）
- [ ] 铅笔、橡皮、削笔刀、直尺
- [ ] 文件袋、姓名贴
- [ ] 水杯、纸巾
- [ ] 固定书桌与台灯

### 家校衔接

- [ ] 关注百旺校区公众号 / 通知
- [ ] 8 月下旬新生家长会（时间以校通知为准）
- [ ] 关注「海淀教育」公众号
- [ ] 加入青龙桥家长群（仅收信息）
