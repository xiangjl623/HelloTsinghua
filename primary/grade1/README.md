## 推荐使用顺序

1. **入学前**：暑假清单 → 语文 / 数学 / 英语素材中的「入学前」章节
2. **9 月**：详细计划 9 月篇 + 周检查表
3. **日常**：按学科素材选活动，不必全部做完——**每周从每科选 2–3 项即可**
4. **选书**：优先海淀图书馆借阅，再考虑购买

---

## 素材使用原则

1. **语文多投、数学按量、英语补信心**——英语不太敢说，**不与其他科比进度**，先听敢说
2. **素材是菜单不是任务**——孩子抗拒的项目可以换同类替代
3. **优先免费资源**——图书馆、公园、社区活动
4. **一年级不报语数外班**——家庭轻量配合即可
