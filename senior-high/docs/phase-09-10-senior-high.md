# 阶段九–十：高中（2035.9 – 2038.7）

**核心任务**：六小强重点班 → 高考 685+ 或强基 / 竞赛 → 清华大学。

---

## 阶段九：高一 – 高二（2035.9 – 2037.7）

### 班型选择

| 学校 | 优先班型 |
|------|----------|
| 清华附 | 理科实验班、AI 相关班型 |
| 人大附 | ICC / 早培延续、理科实验 |
| 十一 | 最高难度分层课程 |

### 选科（北京「3+3」以 2035 年政策为准）

- 清华 90% 以上专业要求 **物理 + 化学**
- 语文强 + 数物化跟上 → 可选清华新雅、未央等交叉书院（强基）

---

## 三条升学轨道

### 轨道 A：高考裸分（主轨道）

| 节点 | 目标 |
|------|------|
| 高二下 | 海淀模考稳定前 1000 |
| 高三模考 | 海淀前 500 |
| 2038 高考 | **685+**，全市前 300–500 |

**适合**：综合强、理科学霸型、语文优势已转化为全科阅读力。

### 轨道 B：强基计划

| 节点 | 动作 |
|------|------|
| 高二下 | 清华强基营 / 学科营 |
| 2038.3 | 强基报名 |
| 2038.6–7 | 校测 + 高考加权 |

- 面向基础学科：数学、物理、化学、生物、历史、哲学等
- 语文强 + 面试表达 → 交叉学科专业有优势

### 轨道 C：竞赛保送（仅天赋型）

| 条件 | 说明 |
|------|------|
| 门槛 | 信息学 / 数学 / 物理 **省队** 级别 |
| 建议 | 无持续省一以上不建议 all-in |
| 投入产出 | 为清华强行走竞赛，性价比低 |

## 综合素质（贯穿高中）

- 独立研究课题
- 公众号 / 写作
- 辩论队
- 持续竞赛 + 论文（若走竞赛轨）

## 阶段十：高三（2037.9 – 2038.7）

### 时间线

| 时间 | 动作 |
|------|------|
| 2037.9 – 2038.1 | 一轮复习；强基政策研究 |
| 2038.3 | 强基报名（若走此通道） |
| 2038.4 – 5 | 二模、三模；志愿模拟 |
| 2038.6.7 – 8 | **高考** |
| 2038.6 – 7 | 出分、填报、录取 |

### 志愿策略

| 情况 | 建议 |
|------|------|
| 685+ | 清华不限组 vs 物理+化学组，按专业倾向选 |
| 差 5–10 分 | 评估北大、复旦、上交、浙大等 |
| 强基校测 | 并行准备面试与学科深度 |

### 家长角色

- 提供后勤与情绪稳定，减少额外压力
- 不反复追问排名
- 志愿共同决策，尊重孩子专业兴趣（在物理+化学框架内）
