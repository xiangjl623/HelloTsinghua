# 双职工家庭 · 周末时间表模板

> 适用：二年级起，至五年级可沿用并微调。  
> 原则：周六偏学习，周日偏语文 + 家庭；每天保留户外。

---

## 周六

| 时段 | 内容 | 时长 |
|------|------|------|
| 8:30 – 9:00 | 早餐、准备 | 30 min |
| 9:00 – 10:30 | **数学思维**（校内同步 + 拓展） | 90 min |
| 10:30 – 10:45 | 休息 | 15 min |
| 10:45 – 11:30 | **英语**（系统课 / 跟读） | 45 min |
| 11:30 – 14:00 | 午餐 + 户外 / 运动 | |
| 14:00 – 15:30 | 自由阅读 / 休息 | |
| 15:30 – 17:00 | 兴趣班（游泳 / 乒乓球，可选） | |
| 晚上 | 家庭时间，21:00 前睡 | |

---

## 周日

| 时段 | 内容 | 时长 |
|------|------|------|
| 8:30 – 9:00 | 早餐 | |
| 9:00 – 10:30 | **语文拓展**（阅读课 / 写作小班） | 90 min |
| 10:30 – 12:00 | 图书馆 / 书店 | |
| 下午 | **家庭日**（博物馆、公园、亲子活动） | |
| 晚上 | 整理下周书包、简单复盘 | |

---

## 平日（周一–周五）

| 时段 | 内容 |
|------|------|
| 放学后 30 min | 休息、加餐 |
| 17:30 – 18:30 | 校内作业（独立完成后家长抽查） |
| 18:30 – 19:00 | 晚餐 |
| 19:00 – 19:30 | 阅读 或 英语磨耳朵（交替） |
| 19:30 – 20:30 | 自由玩耍 / 洗漱 |
| 20:30 – 21:00 | 入睡（三年级后可略延至 21:30） |

---

## 可视化时间表（给孩子）

打印空白表，让孩子自己贴贴纸标记完成情况：

```
周一  周二  周三  周四  周五  |  周六  周日
[作业][作业][作业][作业][作业] | [数学][英语]
[阅读][阅读][阅读][阅读][阅读] | [语文][家庭]
```

---

## 调整说明

| 年级 | 调整 |
|------|------|
| 二年级 | 数学拓展 20 min × 2 次即可，不必满 90 min |
| 三年级 | 英语系统课固定时段 |
| 四–五年级 | 周六可加 30 min 竞赛 / KET/PET 备考 |
| 六年级 | 周六改为综合卷 + 复习，见 [阶段四–六](phase-04-06-grades4-6.md) |

---

## 双职工衔接技巧

1. **老人 / 托管**：明确「先作业后玩」规则，与学校一致。
2. **碎片时间**：接送路上听英文音频；等餐时口算 5 题。
3. **夫妻分工**：一人负责数学英语，一人负责语文阅读，周末互换或固定。
4. **不补课日**：每月留 1 个周六完全休息，防止 burnout。
