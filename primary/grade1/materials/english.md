# 英语培养素材 · 一年级

> **基线**：英语**相对薄弱**；**不太敢说**（开口信心不足）  
> 原则：**先听敢说，再求发音**；磨耳朵为主，**零压力开口**  
> 家庭投入：每日 20 分钟（听 15 + 开口 5，可拆分）  
> 目标：听懂简单指令；**愿意**跟唱 / 跟读；不背词表

## 一、当前基线（2026 年）

| 维度 | 情况 |
|------|------|
| 相对语文 / 数学 | 英语为**主要短板** |
| 开口 | **不太敢说**——听得多、说得少，或一开口就怕错 |
| 家庭策略 | **降门槛**：只听不说也可以；开口从儿歌、TPR、回声句开始 |
| **不做** | 纠音过狠、当众考单词、与英语好的同学比、一年级报班考级 |

**策略**：一年级英语**第一指标是「敢开口」**，不是词汇量或发音标准。与语文「表达超前」形成反差是正常的——**不要用中文表达标准要求英文**。家庭顺序：**大量可理解输入 → 无评判跟唱/跟读 → 再慢慢对话**。

### 不太敢说，怎么破

| 场景 | 做法 |
|------|------|
| 日常 20 min | **前 3 个月以听为主**；孩子愿意再跟唱 1 句即可，不追问 |
| 跟读 | 家长先读，孩子**回声**（重复最后 2–3 词），不必整句 |
| 纠音 | **只夸「听到了」**；发音不准先放过，二年级再细纠 |
| 开口场合 | 仅家庭 / 车内；**不要求**在亲戚、同学前表演英文 |
| 与音乐协同 | 英文儿歌可跟 [music.md](music.md) 班课节奏，降低陌生感 |
| 不做 | 「你怎么不开口」「这都不会说」 |

## 二、学年能力目标（修订）

| 能力 | 一年级末目标 |
|------|--------------|
| 听力 | 听懂日常课堂指令（Stand up, Open your book 等） |
| 开口信心 | **愿意**跟唱 5–10 首儿歌；能跟读 2–3 句简单句（小声即可） |
| 口语 | 家庭 TPR 游戏能响应；**不要求**主动英文对话 |
| 阅读 | 「听读」牛津树 L1 部分册；**不要求**独立读英文 |
| 词汇 | 不背词表；自然积累 200–300 可理解词 |
| **不做** | 背单词、语法课、KET、写英文作业、强迫当众说英文 |

## 三、每日 20 分钟结构（可拆分 · 重听轻说）

| 时段 | 时长 | 内容 |
|------|------|------|
| 输入 | 12–15 min | 动画 / 有声绘本 / 儿歌（**必做**） |
| 开口 | 3–5 min | 跟唱 1 段 **或** 回声 1 句 **或** TPR（**孩子不愿可跳过**） |
| 游戏 | 2 min | 指物、Simon Says（家长说，孩子做，**不必开口**） |

**双职工技巧**：

- 上学路上：儿歌音频
- 洗澡时：播放同一集动画音频
- 睡前：1 本英文绘本

## 四、分级听力资源

### 动画（优先，画面辅助理解）

| 名称 | 难度 | 集长 | 说明 |
|------|------|------|------|
| Peppa Pig 第 1–2 季 | ★ | 5 min | 英音，语速慢 |
| Bluey | ★ | 7 min | 澳音，家庭场景 |
| Maisy Mouse | ★ | 5 min | 词汇极简 |
| Numberblocks | ★★ | 5 min | 数字概念，配合数学 |
| Alphablocks | ★★ | 5 min | 字母音，自然拼读启蒙 |

**观看规则**：开英文字幕可，**不开中文字幕**；同一集反复看 3 天再换。

### 有声绘本 / 分级读物

| 系列 | 级别 | 获取 |
|------|------|------|
| 牛津树 Oxford Reading Tree | L1–3 | 实体书 + 点读笔（可选） |
| 海尼曼 Heinemann GK | GK | 套装或图书馆 |
| 红火箭 Red Rocket | 早期 | 蓝盒前 40 本 |
| 培生幼儿英语 | 预备级 | 常见 |

**一年级目标**：精听 **30–50 本**（反复读 > 泛读百本）。

### 儿歌清单（10 首入门）

1. Twinkle Twinkle Little Star  
2. Old MacDonald Had a Farm  
3. If You're Happy and You Know It  
4. Head Shoulders Knees and Toes  
5. The Wheels on the Bus  
6. Baa Baa Black Sheep  
7. Five Little Monkeys  
8. Row Row Row Your Boat  
9. Baby Shark（孩子通常已会）  
10. Hello Hello How Are You  

**玩法**：边做动作边唱（TPR），每周加 1 首，旧歌复习。

## 五、开口活动（由易到难 · 10 月起逐步加）

> 孩子**不太敢说**时，从 A 做起；A 稳定 2 周再试 B。**不跳级。**

### 活动 A · 只听 + 动（零开口压力）

- 看 Peppa / Bluey：开英文字幕，**不暂停、不提问**
- 儿歌：家长唱，孩子拍手 / 做动作即可
- TPR：家长说 Stand up，孩子站起来——**响应动作即算完成**

### 活动 B · 跟唱（比跟读更容易）

1. 选已熟悉的儿歌（Baby Shark、Twinkle 等）
2. 放音频，家长和孩子一起唱
3. 孩子只唱**最后 1 个词**也算成功（如 …little **star**）

### 活动 C · 影子跟读 / 回声（5 min，**11 月起**）

1. 播放 1 句（动画或绘本音频）
2. 家长先读一遍，孩子**重复最后 2–3 个词**
3. 孩子不愿开口 → 改回活动 B，**不强迫**

**起步句**：

- Hello, I'm Peppa.
- This is my brother George.
- Let's go outside.

### 活动 D · 指物 + 可选说词（3 min）

家中贴英文标签（optional）：

| 中文 | 英文 |
|------|------|
| 门 | door |
| 窗 | window |
| 桌 | table |
| 苹果 | apple |

游戏：「Touch the door!」——孩子**指对即可**；愿意再说 door 更好，不说也 OK。

### 活动 E · TPR 全身反应（配合校内）

| 指令 | 动作 |
|------|------|
| Stand up / Sit down | 站 / 坐 |
| Open / Close your book | 开合书 |
| Listen / Look | 指耳 / 指眼 |
| Clap your hands | 拍手 |
| Touch your nose | 摸鼻子 |

**家长发指令，孩子做**——与 [专注力训练](../../templates/focus-training-games.md) 可合并。

## 六、自然拼读：一年级要不要学？

| 观点 | 本计划建议 |
|------|------------|
| 提前学 Phonics | 不系统上课；通过 Alphablocks、牛津树自然接触 |
| 学校教什么 | 完全跟随，回家听读对应内容 |
| 写字母 | 随校内，家庭不额外练 |

**原则**：听力输入量 >> 拼读规则；**听够 200 小时再系统拼读**（约二年级）。

## 八、按月推荐

| 月份 | 输入 | 开口（低压力） |
|------|------|----------------|
| 8 月 | 儿歌 3 首 + 动画 10 min | **只听**，跟拍即可 |
| 9 月 | Peppa 第 1 季，磨耳朵 | TPR 响应动作 |
| 10 月 | 牛津树 L1 + 儿歌 | 跟唱 1 首（不必读） |
| 11–12 月 | 牛津树 L1–2 | 回声 1 句 / 跟唱（**选一种**） |
| 1–2 月 | 重复精听，不换新 | 巩固已会儿歌 |
| 3–6 月 | 牛津树 L2 + Numberblocks | 回声 2–3 句；仍**不要求**对话 |

## 九、常见误区

| 误区 | 正确 |
|------|------|
| 背 100 个单词 | 在听读中自然习得 |
| 纠音太狠 | **先敢开口**；不太敢说时更要少纠 |
| 中英夹杂讲解 | 看动画时不暂停翻译 |
| 一年级报 KET / 英语班 | 四年级再考虑 KET；一年级**不报**英语班 |
| 写英文作业 | 一年级以听为主 |
| 「你怎么不说英语」 | 听够 100+ 小时再期待主动说 |
| 用中文表达标准要求英文 | 语文超前 ≠ 英语也要超前 |

## 十、与校内衔接

- 入学后问老师：用什么教材？有无配套音频？
- 回家只**重复**学校内容，不加难度
- 若学校暂无英语课：家庭按本素材执行即可
