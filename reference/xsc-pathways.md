# 小升初通道速查 · 中关村二小百旺校区

> 百旺校区**无直升**；六年后主要路径：登记入学 + 青龙桥学区派位。  
> 政策以**当年**海淀区教委文件为准，本表基于 2025–2026 年信息整理。

---

## 一、青龙桥学区派位中学（兜底）

| 学校 | 档次 | 说明 |
|------|------|------|
| 上地实验学校 | ★★★★ | 主目标，教学质量较好 |
| 北京市八一学校 | ★★★★ | 主目标 |
| 人大附中西山学校 | ★★★★ | 主目标 |
| 首师大附中北校区 | ★★★ | 保底优选 |
| 101 中学温泉校区 | ★★★ | 保底优选 |
| 北京市第二十中学 | ★★★ | 保底 |
| 首师大附中科学城学校 | ★★★ | 较新，可期待 |
| 农大附中 | ★★ | **尽量避免**；约 18% 概率 |

**志愿策略**：二派优先填上地实验 → 八一 → 人大附西山 → 首师大附北 → 101 温泉。

## 二、公办初中登记入学（跳出派位）

面向海淀区学籍，电脑派位，**必须填报**。

| 学校 | 近年约招人数 | 备注 |
|------|--------------|------|
| 北京十一学校 | ~150 | 冲刺 |
| 北京一零一中学 | ~140 | 冲刺 |
| 清华大学附属中学 | ~100 | 冲刺；教职工子女另有通道 |
| 中国人民大学附属中学 | ~80 | 冲刺 |
| 北京大学附属中学 | ~80 | 冲刺 |
| 首师大附中 | ~20 | 名额少 |

**准备**：五下起综合卷；六上开放日 / 体验营；数学竞赛奖项提升材料厚度。

## 三、海淀区人工智能实验班

| 项目 | 内容 |
|------|------|
| 面向 | 当年 9 月入学初一的海淀学籍学生 |
| 规模 | 约 180 人 |
| 主办 | 海淀教科院 + 清华计算机系 + 清华附中 |
| 时间 | 通常 7 月网上报名 |
| 选拔 | 自评 + 简历 + 笔试 + 面试 |
| 课程 | 计算思维、数学思维、程序设计、AI 实验等（周六 + 假期） |

**报名网址（以当年通知为准）**：https://ai.hdjky.org.cn/

**准备**：五下 Scratch / Python 启蒙；语文表达与逻辑在面试中有优势。

## 七、若落入一般初中

- 目标：**年级前 5%**，锁定校额到校
- 课外：六小强名师网课 / 小班补难度
- 仍可报 AI 实验班（若小升初时未入选）
- 2035 中考时，六小强有校额到校名额分配给普通初中尖子生
