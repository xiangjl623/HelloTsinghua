# 家长年度待办清单

> 每年 9 月打印勾选；5–7 月额外关注升学政策。

---

## 一至二年级（2026 – 2028）

### 每年必做

- [ ] 参加至少 1 次家长会
- [ ] 每月更新 [成长档案](growth-archive.md)
- [ ] 每日 / 每周亲子阅读（一年级）→ 自主阅读（二年级）
- [ ] 固定作息：21:00 前睡（低年级）
- [ ] 周末：半天户外 + 半天阅读
- [ ] 游泳、跳绳、音乐三项坚持（见 [兴趣总览](../grade1/materials/interests.md)）

### 一年级特有（2026–2027）

- [ ] [专注力训练](focus-training-games.md) 每周 3–4 次
- [ ] 不做奥数、不抢跑课本

### 二年级特有（2027–2028）

- [ ] 启用 [周末时间表](weekly-schedule-dual-income.md)
- [ ] 数学《举一反三》A 版，每周 2–3 次
- [ ] 参加 1 次朗诵 / 讲故事活动

---

## 三至四年级（2028 – 2030）

### 每年必做

- [ ] 评估数学、英语是否与语文同级
- [ ] 记录竞赛 / 证书到成长档案
- [ ] 体育每周 ≥ 2 次
- [ ] 了解 [小升初通道](../reference/xsc-pathways.md)（三年级起）

### 三年级特有（2028–2029）

- [ ] 数学奥数勤思班；英语系统课
- [ ] 非虚构阅读每周 1 本

### 四年级特有（2029–2030）

- [ ] KET 备考 / 报考
- [ ] 希望杯 / 华杯赛体验
- [ ] 作文征文 1 次（叶圣陶杯 / 冰心等）
- [ ] 辩论 / 小记者 / 朗诵择 1 项

---

## 五至六年级（2030 – 2032）

### 每年必做

- [ ] **5 月**：公办初中登记入学政策、填报志愿
- [ ] **6 月**：学区派位志愿
- [ ] **7 月**：人工智能实验班报名（六年级）
- [ ] 综合卷每周 1 套（五年级下起）
- [ ] 更新简历材料（竞赛、英语、作文）

### 五年级特有（2030–2031）

- [ ] PET 备考
- [ ] Scratch / Python 启蒙
- [ ] 海淀科技馆 AI 体验 1 次

### 六年级特有（2031–2032）

- [ ] 目标校开放日 / 体验营
- [ ] 数学竞赛区级以上二等奖+
- [ ] 心理建设：派位结果无论好坏都有路径

---

## 七至九年级 · 初中（2032 – 2035）

### 每年必做

- [ ] 关注期中 / 期末排名（路径 A：前 20%；路径 B：前 5%）
- [ ] 校额到校政策（一般校）
- [ ] 体育中考项目训练（初二起）
- [ ] 小四门背记（初二下起）

### 初二特有（2033–2034）

- [ ] 评估 1+3 是否适合
- [ ] 评估竞赛 vs 中考 + 强基
- [ ] CSP-J 或物理启蒙（若选）

### 初三特有（2034–2035）

- [ ] 中考志愿模拟
- [ ] 4–5 月体育中考
- [ ] 6 月中考

---

## 十至十二年级 · 高中（2035 – 2038）

### 每年必做

- [ ] 选科与班型确认（物理 + 化学为主）
- [ ] 强基 / 竞赛信息跟踪
- [ ] 1 项深度项目持续

### 高二特有（2036–2037）

- [ ] 清华强基营 / 学科营报名
- [ ] 海淀模考位次跟踪

### 高三特有（2037–2038）

- [ ] 3 月强基报名
- [ ] 4–5 月模考与志愿
- [ ] 6 月高考
- [ ] 6–7 月填报

---

## 每年都该做的事

| 时间 | 动作 |
|------|------|
| 9 月 | 读对应 [阶段文档](../README.md#文档导航)，更新年度目标 |
| 1 月 | 上学期复盘，调整周末班 |
| 5–7 月 | 查 [政策信息源](../reference/policy-sources.md) |
| 7 月 | 暑假计划（参考 [2026 模板](summer-2026-checklist.md) 自拟） |

---

## 三条底线（每年重读）

1. 语文优势 ≠ 偏科理由；四五年级数学英语必须拉平。
2. 第一关是小升初，不是高考。
3. 身心健康 > 短期排名。
