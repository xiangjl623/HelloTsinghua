# 阶段七–八：初中（2032.9 – 2035.7）

**核心任务**：优质初中前 20% → 中考进海淀六小强本部。

---

## 阶段七：初一 – 初二（2032.9 – 2034.7）

### 路径 A：进入上地实验 / 八一 / 人大附西山

| 维度 | 目标 |
|------|------|
| 班型 | 争取实验班 / 重点班（入学测 + 期中成绩） |
| 数学 | 继续奥数体系；探索 **CSP-J** 或物理启蒙（二选一） |
| 英语 | FCE / 托福 Junior 800+ |
| 语文 | 保持优势；系统学习初中文言文 |

### 路径 B：进入一般初中

| 维度 | 目标 |
|------|------|
| 排名 | **年级前 5%**，锁定校额到校 |
| 课外 | 六小强名师网课 / 小班补难度 |
| 补充 | 仍可报清华附 AI 实验班 |

### 初二关键节点（2034）

- [ ] 了解 **「1+3」培养实验**（初二升实验高中）—— 慎选，牺牲部分选择自由
- [ ] 评估竞赛路线：持续区级以上数学 / 信息学奖项 → 竞赛；否则 → 中考 + 强基
- [ ] 体育中考准备：跳绳、实心球、800 米，目标满分（40 分）

### 初一–二阶段验收

- [ ] 主科年级前 20%（路径 A）或前 5%（路径 B）
- [ ] 英语 FCE 或同等
- [ ] 体育项目每周 ≥ 2 次

---

## 阶段八：初三（2034.9 – 2035.7）

**核心任务**：中考进六小强本部（至少海淀前 5000，最好前 2000）。

### 目标分数参考（2035 年以当年为准）

| 项目 | 参考 |
|------|------|
| 海淀六小强本部线 | 约 630–650+ / 660（含体育） |
| 对应水平 | 主科约 85–90% 得分率 |

### 分科策略

| 学科 | 策略 |
|------|------|
| 语文（优势） | 确保高分段；作文模板 + 素材库 |
| 数学 | 压轴题突破；错题本每周过 |
| 英语 | 阅读完形不丢分 |
| 小四门 | 初二下开始背记，减轻初三负担 |

### 备选通道

| 通道 | 说明 |
|------|------|
| 校额到校 | 校内排名靠前可降 5–15 分进六小强 |
| 1+3 | 若初二已选，按项目要求执行 |
| 国际部 | 非清华主线，除非明确出国 |

### 初三时间线

| 时间 | 动作 |
|------|------|
| 2034.9 – 2035.1 | 一轮复习；每周模拟卷 |
| 2035.3 – 4 | 二模、三模；志愿策略 |
| 2035.4 – 5 | 体育中考 |
| 2035.6 | 中考 |
| 2035.7 | 出分、录取 |

### 心理支持

- 避免「非六小强不上」的绝对化表述
- 校额到校是普通校尖子的重要通道，提前两年关注校内排名

## 六小强目标校（中考）

| 学校 | 与清华关联 |
|------|------------|
| 清华大学附属中学 | 最近；理科实验、AI 相关班型 |
| 中国人民大学附属中学 | 顶尖理科、ICC |
| 北京十一学校 | 分层走班、高自主 |
| 北京一零一中学 | 综合强校 |
| 北京大学附属中学 | 理科、创新 |
| 首师大附中 | 数学 + AI 方向 |
