# 阶段四：四年级（2029.9 – 2030.7）

**核心任务**：形成「语文引领、数理跟进、英语突破」三角结构。

## 学业重点

| 学科 | 目标 | 做法 |
|------|------|------|
| 语文 | 深度阅读 + 写作框架 | 名著简版、人物传记；记叙文 / 说明文结构 |
| 数学 | 竞赛体验 | 希望杯 / 华杯赛四年级组，不求获奖 |
| 英语 | KET | 词汇 1500+，能写简单段落 |

## 特长（结合语文优势）

- [ ] 辩论 / 朗诵 / 小记者 / 图书馆志愿者（择 1–2）
- [ ] 体育固定 1 项：**乒乓球或游泳**

## 预算（适中档）

- 语数外周末班：3000–5000 元 / 月
- 假期 camp / 研学：1–2 次 / 年，5000–8000 元 / 次

## 阶段验收

- [ ] KET 通过或同等
- [ ] 数学校级竞赛参与奖 / 三等奖
- [ ] 1–2 篇校级以上作文获奖或发表

# 阶段五：五年级（2030.9 – 2031.7）

**核心任务**：数学英语拉到与语文同级；启动小升初信息战。

## 学业重点

| 学科 | 目标 |
|------|------|
| 语文 | 区 / 校前列；科学史、人物传记、社科入门 |
| 数学 | 提高班；迎春杯 / 希望杯，目标**区级以上奖项** |
| 英语 | 冲击 **PET**；原版书（Magic Tree House → 夏洛的网） |

## 小升初通道（按优先级）

详见 [通道速查](../reference/xsc-pathways.md)。

1. **公办初中登记入学**（六小强全区名额）
2. **海淀区人工智能实验班**（180 人，7 月报名）
3. **人大附早培 / 八少八素**（可尝试不押宝）
4. **青龙桥学区二派**（兜底志愿）

## 五年级额外行动

- [ ] 五下起：海淀 RJ / TS 类综合卷，每周 1 套
- [ ] 五下起：Scratch / Python 启蒙，每周 1 次
- [ ] 参加 1 次海淀科技馆 AI / 编程体验

## 阶段验收

- [ ] 数学区级以上竞赛有奖项
- [ ] PET 通过
- [ ] 完成 1 次 AI / 编程体验项目

# 阶段六：六年级（2031.9 – 2032.7）

**核心任务**：小升初决战——进清华路线的**第一关**。

## 时间线

| 时间 | 动作 |
|------|------|
| 2031.9 – 2032.1 | 语数英三轮复习；每周 1 套综合卷 |
| 2032.3 – 4 | 目标初中开放日；投递简历（如有） |
| 2032.5 | 公办初中登记入学填报志愿 |
| 2032.6 | 学区派位志愿填报 |
| 2032.7 | 人工智能实验班报名 + 选拔 |

## 学业目标

| 学科 | 目标 |
|------|------|
| 语文 | 区统测前 10%；作文有个人风格 |
| 数学 | 迎春杯 / 希望杯**二等奖以上** |
| 英语 | PET 优秀或 FCE 入门 |

## 目标初中分档

| 档位 | 学校 |
|------|------|
| 冲刺 | 清华附、人大附、十一、101 |
| 主目标 | 上地实验、八一、人大附西山 |
| 保底 | 首师大附北、101 温泉、二十中 |

## 心理建设话术

> 「我们尽力选最好的学校，但无论去哪，你的阅读能力都是最大财富。」

若落入一般初中：启动**校额到校**意识（2035 中考用）。

## 六年级周末调整

| 周六 | 周日 |
|------|------|
| 综合卷 1 套 + 错题复盘 | 语文作文 + 开放日 / 访校 |
| 数学竞赛题巩固 | 英语阅读 + 家庭休息 |
