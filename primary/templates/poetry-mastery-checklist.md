| 1 | | | ☐ |
| 2 | | | ☐ |
| 3 | | | ☐ |
| 4 | | | ☐ |

**原则**：已背 > 新增；优先复习二、三年级已勾「已背」篇目，防遗忘；一年级课内篇目随校内进度再勾。
