# 阶段二：二年级（2027.9 – 2028.7）

**核心任务**：把「会学习」变成自动程序。

## 学业重点

| 学科 | 目标 | 具体做法 |
|------|------|----------|
| 语文 | 自主阅读 | 从亲子阅读过渡；写 3–5 句日记，每周 2 次 |
| 数学 | 浅层思维 | **二年级**《举一反三》A 版，每周 2–3 次 × 20 分钟（入学前已完成一年级 A 版） |
| 英语 | 跟读对话 | 剑桥少儿或校内拓展；不追求考级 |

## 能力培养

- **时间管理**：可视化时间表，放学后 1.5h 学习区块 → 见 [周末时间表](../../shared/templates/weekly-schedule-dual-income.md)
- **错题意识**：数学「做一遍 → 讲一遍」（费曼学习法雏形）
- **表达训练**：参加班级 / 校级朗诵、讲故事

## 阶段验收

- [ ] 作业无需家长盯着完成
- [ ] 语文稳居班级前列
- [ ] 数学无明显畏难
- [ ] 英语能跟读简单段落

# 阶段三：三年级（2028.9 – 2029.7）

**核心任务**：海淀「分水岭」起点——偏科在此显现，数学英语必须补强。

## 学业重点

| 学科 | 目标 | 具体做法 |
|------|------|----------|
| 语文 | 跨学科阅读 | 非虚构类（科普、历史故事）；阅读理解答题方法 |
| 数学 | **关键补强** | 学而思 / 高思**勤思班**（非最高班）；重逻辑不重竞赛 |
| 英语 | 系统课 | 每周 1–2 次；三年级末能读简单章节书 |

## 升学意识（轻量启动）

- [ ] 了解青龙桥学区派位中学 → [通道速查](../reference/xsc-pathways.md)
- [ ] 加入 1–2 个海淀家长信息群，**只收政策不攀比**

## 派位中学（提前知晓）

上地实验、八一、人大附西山、101 温泉、首师大附北、二十中、农大附中、首师大附科学城等。

## 阶段验收

- [ ] 语数英校测稳定班级前 30%
- [ ] 每周阅读 ≥ 3 本（含 1 本非虚构）
